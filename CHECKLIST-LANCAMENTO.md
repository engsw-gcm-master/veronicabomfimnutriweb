# ✅ CHECKLIST DE LANÇAMENTO DO SITE

Use este checklist para garantir que tudo está pronto antes de publicar.

---

## PRÉ-LANÇAMENTO (Antes de Publicar)

### Verificação de Conteúdo
- [ ] Título principal correto
- [ ] Descrição sobre você atualizada
- [ ] Foto de perfil aparece
- [ ] Número de WhatsApp correto
- [ ] Email correto
- [ ] Instagram atualizado
- [ ] Todos os serviços listados
- [ ] Pelo menos 3 receitas adicionadas
- [ ] Pelo menos 3 posts de blog
- [ ] Seção de contato completa

### Verificação Visual
- [ ] Cores conferem com guia de marca
- [ ] Fontes Playfair Display em títulos
- [ ] Fontes TT Norms em corpo
- [ ] Logo aparece na navegação
- [ ] Sem erros de ortografia (leia tudo!)
- [ ] Links funcionam (clique em todos)
- [ ] Botões respondem ao clique
- [ ] Menu mobile funciona

### Teste Responsivo
- [ ] Teste em celular (real ou emulado)
- [ ] Teste em tablet
- [ ] Teste em PC grande
- [ ] Teste em navegador Firefox
- [ ] Teste em Chrome
- [ ] Teste em Safari
- [ ] Sem scroll horizontal
- [ ] Botões grandes para tocar

### Performance
- [ ] Página carrega em < 3 segundos
- [ ] Imagens aparecem rápido
- [ ] Sem muitos erros no console (F12)
- [ ] Sem avisos de segurança

### SEO Básico
- [ ] Meta description preenchida
- [ ] Título da página correto
- [ ] Meta keywords adicionadas
- [ ] Schema markup presente
- [ ] Alt text em imagens
- [ ] H1 único e descritivo
- [ ] Links internos funcionando

### Formulários & Interação
- [ ] Formulário de contato funciona
- [ ] Botão WhatsApp redireciona corretamente
- [ ] Links do Instagram funcionam
- [ ] Newsletter (se houver) funciona
- [ ] Sem campos obrigatórios impossíveis

---

## PUBLICAÇÃO (No Cloudflare)

### Upload
- [ ] Criou conta Cloudflare
- [ ] Efetuou upload de todos os arquivos
- [ ] Verificou se não esqueceu de pasta
- [ ] Site recebeu URL (seu-site.pages.dev)

### Teste Pós-Publicação
- [ ] Acessou o link do site
- [ ] Site carrega completamente
- [ ] Foto aparece
- [ ] Menu funciona
- [ ] Responsivo no celular
- [ ] Links externos funcionam
- [ ] WhatsApp redireciona

### Configuração Cloudflare
- [ ] Dashboard acessível
- [ ] Analytics ativado
- [ ] SSL/HTTPS ativo (automático ✓)
- [ ] Cache habilitado (automático ✓)

## PÓS-LANÇAMENTO (Depois de Publicar)

### Google & SEO
- [ ] Enviou sitemap para Google Search Console
- [ ] Criou Google Business Profile
- [ ] Adicionou endereço e horários
- [ ] Criou conta Google Analytics
- [ ] Instalou código GA no site

### Redes Sociais
- [ ] Atualizou bio Instagram (link para site)
- [ ] Atualizou foto de perfil
- [ ] Atualizou descrição
- [ ] Compartilhou link no feed
- [ ] Compartilhou nos stories
- [ ] Adicionou link no WhatsApp status

### Marketing Inicial
- [ ] Enviou site para todos contatos por WhatsApp
- [ ] Postou no Instagram feed
- [ ] Postou no Instagram stories
- [ ] Postou em grupos de WhatsApp
- [ ] Pediu para amigos compartilharem

### Backup & Segurança
- [ ] Guarde cópia da pasta no PC
- [ ] Se usar GitHub, faça primeiro push
- [ ] Teste restauração de backup (importante!)
- [ ] Anotou credenciais Cloudflare
- [ ] Anotou credenciais Google

- [ ] Revise analytics
- [ ] Conta visitantes
- [ ] Conta agendamentos via site
- [ ] Atualiza conteúdo obsoleto
- [ ] Adiciona novo artigo/receita
- [ ] Muda foto ou stories

**Instagram:**
- [ ] 12-16 posts no mês
- [ ] 3-5 reels
- [ ] Stories diários
- [ ] Interage com seguidores

**Email/Newsletter:**
- [ ] 2 newsletters enviadas
- [ ] Artigos/receitas compartilhadas

## RED FLAGS (Coisas que Indicam Problema)

❌ Site offline/erro 404
❌ Foto não carrega
❌ Menu desapareceu
❌ Cores muito diferentes
❌ WhatsApp quebrado
❌ Muitos erros no console
❌ Site muito lento
❌ Responsivo ruim no celular

**Se encontrar**: Verifique se todos arquivos foram enviados e nomes estão corretos.
